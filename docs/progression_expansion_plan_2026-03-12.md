# Progression Expansion Plan

## Phase 1 Completed

- 광고 시청으로 하루 동안 여는 `확장 리포트`
- 비전투 상태 HP 자연 회복
- 레벨업 시 `3포인트 자동 성장 + 나머지 수동 분배`
- 퀘스트 카테고리 기반 성장 데이터 수집

## Phase 2

### Monthly / Yearly Quest Hierarchy

- `QuestType.monthly`, `QuestType.yearly` 추가
- 월간/연간 퀘스트는 일반 퀘스트와 분리해 `레이드` 성격으로 노출
- 보상은 일반 XP보다 `칭호`, `프레임`, `대량 골드`, `특수 리포트 해금` 중심으로 설계
- 진행도는 `n / target` 구조로 보여주고, 세부 하위 목표를 둘 수 있게 확장

### Growth Weight Upgrade

- 현재는 완료 퀘스트 XP 기반 가중치
- 다음 단계에서는 `난이도`, `연속 수행`, `주간 퀘스트` 보정치 강화
- 예시:
  - 일일: 1.0x
  - 주간: 1.5x
  - 월간: 2.5x
  - 연간: 4.0x

## Phase 3

### Avatar / Characterization

- 회원가입 단계에는 넣지 않음
- 프로필 또는 상태창에서 `기본 프리셋 아바타 선택` 먼저 제공
- 이후 `헤어`, `의상`, `오라`, `프레임` 커스터마이징 추가
- 사용자 사진 기반 캐릭터화는 마지막 단계로 미룸

## UX Notes

- 자동 성장 결과는 레벨업 모달에서 반드시 보여줘야 함
- 월간/연간 목표는 탭 추가보다 `레이드` 또는 `장기 목표` 섹션으로 분리하는 편이 좋음
- 확장 리포트는 광고 시청 후 `오늘 하루 해금` 규칙을 유지
