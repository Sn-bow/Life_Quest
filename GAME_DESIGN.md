## 1. 컨셉 & 비전

### 핵심 컨셉
> "현실에서 쌓은 능력이 카드가 되고, 그 카드로 던전을 정복한다"

Life Quest 앱의 사용자는 현실의 일상 퀘스트(운동, 공부, 건강관리 등)를 수행하며 경험치와 스탯을 얻는다. 이 성장이 **카드 덱**으로 변환되어, 로그라이크 던전에서 전략적 전투를 즐길 수 있다.

### 게임의 역할
- ❌ 앱의 주(主) 기능이 아님
- ✅ 현실 퀘스트 수행의 **보상이자 동기부여 도구**
- ✅ 장기적으로 확장 가능한 **핵심 엔터테인먼트 콘텐츠**

### 디자인 원칙
1. **현실 우선**: 게임만으로는 강해질 수 없음. 현실 퀘스트가 유일한 성장 경로
2. **짧은 세션**: 1런 5~10분. 출퇴근 중에도 가능
3. **깊은 전략**: 매 턴, 매 노드에서 의미 있는 선택
4. **무한 리플레이**: 로그라이크 + 일일 던전으로 매일 다른 경험
5. **확장 가능**: 카드/몬스터/존/렐릭 추가만으로 콘텐츠 확장

## 2. 현실 연동 시스템

### 2.1 스탯 → 게임 효과 매핑

| 현실 스탯 | 카드 계열 | 게임 내 효과 |
|---|---|---|
| **근력 (STR)** | 🗡️ 공격 카드 | 공격 카드 기본 데미지 +STR×0.5, 공격 카드 등장 확률↑ |
| **지혜 (WIS)** | 🔮 마법 카드 | 턴당 추가 드로우 (WIS 30 이상: +1), 마법 카드 데미지 +WIS×0.3 |
| **건강 (HP)** | 🛡️ 방어 카드 | 시작 HP +건강×2, 방어 카드 방어값 +HP×0.3 |
| **매력 (CHA)** | ✨ 전술 카드 | 상점 할인 CHA×1%, 희귀 카드 등장률 +CHA×0.2% |

### 2.2 퀘스트 완료 → 카드 획득 흐름

```
현실 퀘스트 완료
    ↓
스탯/경험치/골드 획득 (기존 시스템 유지)
    +
카드 포인트(CP) 획득
    ↓
CP 일정량 모이면 → 카드 팩 생성
    ↓
카드 팩 오픈 → 3장 중 1장 선택
    ↓
내 "카드 컬렉션"에 추가
```

### 2.3 카드 포인트(CP) 획득량

| 퀘스트 타입 | 기본 CP | 비고 |
|---|---|---|
| 일일 퀘스트 | 1~3 CP | 난이도에 비례 |
| 주간 퀘스트 | 5~10 CP | |
| 월간 퀘스트 | 20~30 CP | 희귀 카드 팩 가능 |
| 연간 퀘스트 | 50~100 CP | 전설 카드 팩 보장 |

**카드 팩 필요 CP:**
- 일반 팩 (Common~Uncommon): 10 CP
- 고급 팩 (Uncommon~Rare): 30 CP
- 전설 팩 (Rare~Legendary): 100 CP

### 2.4 퀘스트 카테고리 → 카드 계열 연동

| 퀘스트 카테고리 | 제공 카드 팩 계열 |
|---|---|
| 근력 카테고리 (운동 등) | 🗡️ 공격 카드 위주 팩 |
| 지혜 카테고리 (공부 등) | 🔮 마법 카드 위주 팩 |
| 건강 카테고리 (건강관리 등) | 🛡️ 방어 카드 위주 팩 |
| 매력 카테고리 (사교 등) | ✨ 전술 카드 위주 팩 |

> "위주"란: 해당 계열 카드 60% + 다른 계열 카드 40% 확률로 등장

### 2.5 장비 → 던전 시작 보너스

기존 장비 시스템과 연동:
| 장비 | 던전 효과 |
|---|---|
| 무기 장착 | 시작 시 공격 카드 1장 추가 드로우 |
| 방어구 장착 | 시작 HP +방어구 방어력×2 |
| 악세서리 장착 | 시작 시 렐릭 1개 추가 선택지 |

### 2.6 연속 출석(스트릭) 보너스

| 스트릭 | 던전 보너스 |
|---|---|
| 3일 연속 | 던전 시작 시 골드 +30 |
| 7일 연속 | 던전 시작 시 카드 선택 1회 추가 |
| 14일 연속 | 던전 시작 시 렐릭 1개 무료 |
| 30일 연속 | 일일 던전 특별 보스 해금 (고급 보상) |

## 3. 카드 시스템

### 3.1 카드 기본 구조

```
CardData {
  id: String              // 고유 ID (예: "atk_strike")
  name: String            // 카드 이름 (예: "강타")
  category: CardCategory  // ATTACK, MAGIC, DEFENSE, TACTICAL
  rarity: CardRarity      // COMMON, UNCOMMON, RARE, LEGENDARY
  cost: int               // 에너지 비용 (0~5)
  description: String     // 효과 설명
  effects: List<CardEffect>  // 실제 효과 목록
  upgradeId: String?      // 업그레이드된 버전 ID
  spritePath: String      // 카드 일러스트 경로
  isUpgraded: bool        // 업그레이드 여부
}
```

### 3.2 카드 효과 타입

```dart
enum CardEffectType {
  damage,           // 적에게 데미지
  block,            // 방어도 획득
  heal,             // HP 회복
  drawCard,         // 카드 드로우
  gainEnergy,       // 에너지 획득
  applyBuff,        // 자신에게 버프
  applyDebuff,      // 적에게 디버프
  aoe,              // 전체 데미지 (다중 적 전투용)
  conditional,      // 조건부 효과
  exhaust,          // 사용 후 소멸 (디스카드 안 됨)
  retain,           // 턴 넘겨도 패에 유지
  innate,           // 항상 첫 손에 잡힘
}
```

### 3.3 상태 효과 (Status Effects)

**디버프 (적에게 부여)**
| 상태 | 효과 | 지속 |
|---|---|---|
| 취약 (Vulnerable) | 받는 데미지 50% 증가 | N턴 |
| 약화 (Weak) | 가하는 데미지 25% 감소 | N턴 |
| 독 (Poison) | 매 턴 N 데미지, 턴마다 1 감소 | 스택 |
| 화상 (Burn) | 매 턴 N 고정 데미지 | N턴 |
| 빙결 (Freeze) | 다음 행동 1회 스킵 | 1회 |

**버프 (자신에게 부여)**
| 상태 | 효과 | 지속 |
|---|---|---|
| 힘 (Strength) | 공격 데미지 +N | 전투 동안 |
| 민첩 (Dexterity) | 방어도 +N | 전투 동안 |
| 가시 (Thorns) | 피격 시 공격자에게 N 데미지 | 전투 동안 |
| 재생 (Regen) | 매 턴 N HP 회복 | N턴 |
| 집중 (Focus) | 마법 카드 데미지 +N | 전투 동안 |
| 불굴 (Fortify) | 턴 넘길 때 방어도 유지 | N턴 |

## 4. 전투 시스템

### 4.1 턴 진행 흐름

```
[턴 시작]
    ↓
에너지 충전 (기본 3)
카드 드로우 (기본 5장)
버프/디버프 처리 (독 데미지, 재생 등)
    ↓
[플레이어 행동]
카드를 에너지를 소모하여 사용
원하는 만큼 카드 사용 가능 (에너지 허용 범위)
    ↓
[턴 종료 버튼]
    ↓
남은 패 → 디스카드 파일로
방어도 리셋 (불굴 버프 없으면)
    ↓
[적 행동]
적이 의도(Intent)에 따라 행동
    ↓
[적 턴 종료]
    ↓
다음 턴으로 → [턴 시작]
```

### 4.2 전투 기본 수치

| 항목 | 기본값 | 비고 |
|---|---|---|
| 시작 에너지 | 3 | 렐릭/스킬로 증가 가능 |
| 턴당 드로우 | 5장 | 지혜 30+ 일 때 +1장 |
| 시작 HP | 80 + (건강 스탯 × 2) | |
| 덱 최소 장수 | 10장 (시작 덱) | |
| 덱 최대 장수 | 제한 없음 | 단, 덱이 커지면 핵심 카드 뽑기 어려움 |
| 디스카드 파일 소진 시 | 셔플하여 드로우 파일로 | |

### 4.3 적 의도(Intent) 시스템

적은 매 턴 시작 시 다음 행동을 **아이콘으로 예고**:
| 아이콘 | 의도 | 설명 |
|---|---|---|
| ⚔️ | 공격 | N 데미지 예정 (숫자 표시) |
| ⚔️⚔️ | 다중 공격 | N 데미지 ×M회 |
| 🛡️ | 방어 | 방어도 획득 예정 |
| 💪 | 버프 | 자기 강화 예정 |
| 💀 | 디버프 | 플레이어에게 디버프 예정 |
| ❓ | 알 수 없음 | 일부 강력한 적은 의도를 숨김 |

### 4.4 다중 적 전투

- 최대 3마리 동시 전투 가능
- 적 대상 선택: 카드 사용 시 탭하여 타겟 지정
- AOE 카드: 전체 대상
- 적 각각 독립적 HP/의도/상태

### 4.5 전투 보상

| 전투 유형 | 골드 | 카드 선택 | 기타 |
|---|---|---|---|
| 일반 전투 | 10~25 | 3장 중 1장 | - |
| 엘리트 전투 | 30~50 | 3장 중 1장 | 렐릭 1개 |
| 보스 전투 | 50~100 | 3장 중 1장 (레어 보장) | 보스 렐릭 1개 + 카드 팩 |

- 엘리트는 최대 2개
- 상점/휴식은 각각 최소 1개 보장
- 노드 간 연결: 상위 층에서 하위 층으로 1~3개 경로

- **매일 자정에 새 던전 생성** (시드 기반)

  - 화: 마법 카드만 등장

### 8.2 공통 이벤트 (모든 존)

#### E01: 수상한 상인
> 길을 걷다 수상한 상인을 만났다. 뭔가 좋은 물건이 있을 것 같은데...
- **A) 물건을 산다** (골드 50) → 랜덤 렐릭 1개
- **B) 카드를 산다** (골드 30) → 레어 카드 1장 선택
- **C) 무시한다** → 아무 일 없음

#### E02: 신비한 샘
> 빛나는 샘을 발견했다. 묘한 힘이 느껴진다.
- **A) 물을 마신다** → HP 25% 회복
- **B) 카드를 담근다** → 랜덤 카드 1장 업그레이드
- **C) 동전을 던진다** (골드 20) → 50% HP 완전 회복, 50% 저주 1장

#### E03: 잠자는 모험가
> 쓰러진 모험가를 발견했다. 도와줄 수 있을 것 같다.
- **A) 도와준다** (HP 10 소모) → 카드 1장 선물 (언커먼+)
- **B) 가방을 뒤진다** → 골드 30~60, 50% 저주 1장
- **C) 지나친다** → 아무 일 없음

#### E04: 저주받은 제단
> 어두운 제단 위에 빛나는 카드가 놓여있다.
- **A) 카드를 가져간다** → 레어 카드 1장 + 저주 1장
- **B) 기도한다** → 저주 1장 제거 (있으면), 없으면 HP 10 회복
- **C) 파괴한다** → 골드 25

#### E05: 카드 제거 기회
> 방랑하는 현자가 덱을 정리해주겠다고 한다.
- **A) 카드 1장 제거** → 덱에서 선택하여 제거
- **B) 카드 1장 업그레이드** → 선택하여 업그레이드
- **C) 카드 변환** → 1장 제거 후 같은 비용의 랜덤 카드 획득

#### E06: 대장장이
> 실력 좋은 대장장이를 만났다.
- **A) 카드 업그레이드** (골드 40) → 카드 2장 업그레이드
- **B) 장비 강화** → 이번 런 공격 +2 or 방어 +2
- **C) 그냥 간다** → 아무 일 없음

#### E07: 도박꾼의 텐트
> 도박꾼이 게임을 제안한다.
- **A) 받아들인다** (골드 30) → 50% 골드 2배 반환, 50% 잃음
- **B) 큰 판** (골드 80) → 40% 골드 3배, 60% 잃음
- **C) 거절한다** → 아무 일 없음

#### E08: 고대의 도서관
> 먼지 쌓인 도서관에서 세 권의 책을 발견했다.
- **A) 공격의 서** → 공격 카드 3장 중 1장 선택 (레어 보장)
- **B) 방어의 서** → 방어 카드 3장 중 1장 선택 (레어 보장)
- **C) 마법의 서** → 마법 카드 3장 중 1장 선택 (레어 보장)

## 9. 진행 & 성장 시스템

### 9.1 메타 진행 (런 간 유지되는 것들)

**영구 해금:**
- 카드 컬렉션 (도감)
- 렐릭 컬렉션 (도감)
- 존 해금 진행
- 등정(Ascension) 레벨
- 업적/칭호

**런마다 리셋:**
- 현재 덱
- 보유 렐릭
- HP
- 골드 (런 내 골드)
- 버프/디버프

### 9.2 던전 클리어 보상 (런 종료 시)

| 결과 | 보상 |
|---|---|
| 보스 클리어 | XP +(존 레벨 × 50), 골드 +(런 내 골드의 30%→계정), 카드 팩 1개 |
| 엘리트에서 패배 | XP +(진행 노드 × 10), 골드 +(런 골드의 10%) |
| 일반 전투에서 패배 | XP +(진행 노드 × 5) |
| 등정 클리어 | 추가 XP ×(등정 레벨 + 1), 특별 칭호 |

### 9.3 기존 시스템과의 연동

```
[캐릭터 레벨]
  → 존 해금 조건
  → 시작 덱 슬롯 수
  → 일일 던전 시도 횟수

[4대 스탯]
  → 카드 기본 위력 보정
  → 시작 HP/에너지/드로우 보정
  → 카드 등장 확률 보정

[장비]
  → 시작 보너스 (카드/HP/렐릭)

[골드]
  → 던전 내 상점 사용 (기존 골드 일부 반입 가능)

[칭호]
  → 던전 시작 시 추가 보너스

[업적]
  → 새 던전 업적 카테고리 추가

[스킬 트리]
  → 기존 스킬을 카드 시스템으로 전환
    (예: "화염 검격" 스킬 보유 → 던전 시작 덱에 "화염 검격" 카드 포함)
```

### 9.4 스킬 트리 → 카드 연동

기존 스킬 트리에서 배운 스킬이 시작 덱에 자동 포함:

### 12.3 Flame ↔ Flutter 통합 패턴

```dart
// card_battle_screen.dart
class CardBattleScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Stack(
        children: [
          // 🔥 Flame 레이어 (배경, 스프라이트, 이펙트)
          GameWidget<BattleGame>(
            game: BattleGame(),
            overlayBuilderMap: {
              'combat_ui': (context, game) => _buildCombatUI(context, game),
            },
            initialActiveOverlays: const ['combat_ui'],
          ),
        ],
      ),
    );
  }

  Widget _buildCombatUI(BuildContext context, BattleGame game) {
    // Flutter 위젯으로 카드 핸드, 에너지, HP 등 표시
    // Provider.of<CardCombatState> 사용
    return Consumer<CardCombatState>(
      builder: (context, combatState, _) {
        return Column(
          children: [
            _topBar(combatState),       // 에너지, 적 정보
            const Spacer(),
            _cardHand(combatState),     // 카드 핸드 (Flutter)
            _bottomBar(combatState),    // 덱/디스카드 카운트, 턴 종료
          ],
        );
      },
    );
  }
}
```

## 15. 확장 계획 (Post-Launch)

### 15.1 단기 확장 (출시 후 1~3개월)

- **새 존 추가** (Zone 6~10): 존 추가만으로 수십 시간 콘텐츠
- **새 카드 시즌**: 월간 새 카드 10~20장 추가
- **이벤트 던전**: 기간 한정 특별 던전 (할로윈, 크리스마스 등)
- **일일 도전**: 매일 다른 조건의 도전 과제

### 15.2 중기 확장 (3~6개월)

- **클래스 시스템**: 전사/마법사/도적/성직자 → 각 클래스별 고유 카드
- **2번째 캐릭터 클래스**: 시작 덱/고유 카드가 다른 새 플레이어블 캐릭터
- **보스 러시 모드**: 보스만 연속 전투
- **카드 합성**: 같은 카드 3장 → 상위 레어리티 카드

### 15.3 장기 확장 (6개월+)

- **PvP 비동기 대전**: 덱 vs 덱 자동 대전 (리더보드)
- **길드 시스템**: 길드 보스 레이드 (기여도 합산)
- **시즌 시스템**: 시즌마다 리셋 + 새 카드/보상
- **스토리 모드**: 각 존별 스토리 + 특수 이벤트 체인
- **커스텀 카드 제작**: 유저가 자신만의 카드 이름/효과 설정 (제한된 범위)

### 15.4 수익화 확장

- **시즌 패스**: 무료/프리미엄 보상 트랙
- **카드 팩 IAP**: 프리미엄 카드 팩 (밸런스 유지, Pay-to-Collect)
- **코스메틱**: 카드 뒷면 스킨, 특별 이펙트, 플레이어 스프라이트 스킨
- **광고 보상**: 광고 시청 → 추가 던전 시도, 추가 카드 선택지
